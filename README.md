## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/Ehabejoel

# Navigate to directory
cd daily-quotes

# Install dependencies
npm install

# Start the app
npm start
```

## 🛠️ Technologies

- Electron
- JavaScript
- HTML/CSS
- Quotes API (api-ninjas.com)

## 💻 Development

```bash
# Run in development mode
npm start

# Build the app
npm run make
```
